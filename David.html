David's Mighty Men

A Unreal Engine Combat Mechanics and UI Demonstration

This project served as a deep dive into game development, focusing on implementing and polishing responsive, state-based combat mechanics and core User Interface (UI) systems using the Unreal Engine.

Project Overview: Combat System & UI

This demonstration showcases a comprehensive, third-person action combat system and foundational game menus developed using the Unreal Engine. The project highlights the integration of responsive, fluid combat mechanics—including combo attacks, input delay management, dodging, and shielding—alongside user-friendly menu navigation for settings management.

Skills Demonstrated:

  • Game Engine Proficiency: Deep knowledge of the Unreal Engine and its Blueprints visual scripting language for gameplay programming.
  • User Interface Design: Creation and implementation of functional Main Menu, Audio Settings, and Video Settings screens.
  • State Machine Implementation: Designed and implemented a robust character state machine to manage animations, player input, and combat transitions.
  • Input and Combat Management: Handled complex player inputs for attack combos, timing windows, and defensive mechanics like dodging and shielding.
  • Version Control: Utilized Git/GitHub for project management, collaboration, and asset tracking.
