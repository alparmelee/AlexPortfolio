David's Mighty Men

A Unreal Engine Combat Mechanics and UI Demonstration

This project demonstrates implementation of responsive, state-based combat mechanics and core User Interface (UI) systems using the Unreal Engine.

I. Combat Mechanics

Mechanics Implemented:

  • Character State Machine: Manages transitions between movement, attacking, and defense.
  • Quick Attack Combo: Chain of light attacks.
  • Strong Attack Combo: Chain of heavy attacks.
  • Attack Combo Linking: Ability to mix quick and strong attacks in a single combo.
  • Input Buffer: Manages player input timing for responsive combo execution.
  • Combo Cooldown/Delay: Enforces a delay after a full combo to prevent spamming.
  • Dodge: Defensive move with invulnerability frames (i-frames).
  • Shield/Block: Defensive stance that mitigates damage.
  • Camera Lock-On: System to maintain camera focus on an enemy target.

II. User Interface (UI) Development

User Interface Elements Implemented (UMG):

  • Modular Menu Architecture: All screens built as reusable UMG widgets.
  • Main Menu: Includes Start Game, Continue, Settings, Credits, and Quit buttons.
  • Video Settings Screen: Functional controls for Screen Resolution and Graphics Quality Presets.
  • Audio Settings Screen: Functional sliders for controlling Master Volume, Music Volume, and SFX Volume.
  • Input Agnostic Navigation: Full functionality with both mouse and gamepad input.
